🍽️ Food Waste Management System

A modern and responsive web application built using HTML and CSS. This project helps users track food waste, manage leftovers, and encourage food donation with a clean and simple user interface.

✨ Features

🥗 Food Waste Tracking – Add and monitor leftover food items
🍛 Donate Food – Encourage sharing surplus food with others
💡 Smart Suggestions – Get ideas to reuse leftover food
📱 Responsive Design – Works smoothly on mobile and desktop
🎨 Clean UI – Simple and user-friendly interface

🛠️ Technologies Used
HTML5
CSS3
🚀 How to Run the Project

Clone the repository

Open the folder

Run the project:

Double-click index.html
OR
Open with any web browser
🎯 Functionality

➕ Enter food item and quantity → Click Add
📊 Tracks basic food waste input (UI level)
🧭 Navigate using menu → Access different sections
📱 Open in mobile/desktop → Responsive layout adapts automatically
